3. Core Calculations 
 
Total Assets = Domestic Assets + Overseas Assets 
Total Liabilities = Mortgage + Personal + Business + Credit + Other 
Net Worth = Total Assets − Total Liabilities 
 
Asset Allocation: 
Allocation % = Asset Category / Total Assets × 100    (Output need to be in Pie charts) 
Note: If Allocation on a particular asset > 60% → Concentration Risk 
 
4. Growth Rate Calculation 
 
Risk Returns: 
Conservative = 5% 
Moderate = 7% 
Aggressive = 10% 
 
Style Returns: 
Income = 4% 
Balanced = 6% 
Growth = 8% 
Speculative = 12% 

 

We calculate return using weighted influence of: 

Risk Appetite (major driver) 

Investment Style (secondary driver) 

Core Formula: 

r = (Risk Weight × Risk Return) + (Style Weight × Style Return) 

Final Calculation: 

Example: 

Risk = Moderate (7%) 
Style = Growth (8%) 

Risk weight= 70% 

Style weight= 30% 

r = (0.7 × 7%) + (0.3 × 8%) 

= 4.9% + 2.4% 

= 7.3% 

Backend Logic: 

risk_returns = { 

    "conservative": 0.05, 

    "moderate": 0.07, 

    "aggressive": 0.10 

} 

style_returns = { 

    "income": 0.04, 

    "balanced": 0.06, 

    "growth": 0.08, 

    "speculative": 0.12 

} 

risk_weight = 0.7 

style_weight = 0.3 

 

r = (risk_weight * risk_returns[risk]) + \ 

    (style_weight * style_returns[style]) 

Once you get the then you can calculate Future Wealth at 65 and Succession at 90. 

 
5. Future Wealth Calculation 
 
At Age 65: 
Years = Retirement Age − Current Age 
Future Wealth = Net Worth × (1 + r)^Years 
 
At Age 90: 
Years = 90 − Current Age 
Estate Value = Net Worth × (1 + r)^Years 

Future Value (at 65) = Present Value × (1 + r)^n 

Where: 

Present Value = Current Net Worth 
r = Expected Annual Return 
n = Years=retirement age-Current age 

 

Example 

Current net worth=4.69M 

Return= 8% 

Years=30

6. Estate Tax Calculation (US Model)

Net to Heirs = Estate Value − Estate Tax
